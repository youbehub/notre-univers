📌 RÉSUMÉ DU PROJET
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Notre Univers est une application web privée, romantique et interactive
destinée à un couple. C'est un espace magique pour:

✨ Partager votre amour
💌 Écrire des messages
📸 Documenter vos souvenirs
🎮 Jouer ensemble
🎵 Créer votre playlist
🎁 Découvrir des surprises


🎯 CARACTÉRISTIQUES PRINCIPALES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Design Ultra-Moderne
   • Thème Galaxy + Luxury (noir, or, violet, bleu)
   • Étoiles animées en arrière-plan
   • Glassmorphism & animations fluides
   • Effet WOW à chaque interaction

✅ 100% Privé & Sécurisé
   • Page de login avec mot de passe
   • Données sauvegardées localement (pas de serveur)
   • Aucune donnée personnelle envoyée

✅ 6 Sections Complètes
   🏠 Accueil        - Message romantique & compteur de jours
   💌 Messages       - Ajouter/afficher messages cachés
   📸 Souvenirs      - Galerie de photos + descriptions
   🎮 Quiz           - Quiz amoureux interactif
   🎵 Playlist       - Votre musique avec signification
   🎁 Surprise       - Défis, messages, idées aléatoires

✅ Fonctionnalités Interactives
   • Animations fluides et transitions
   • Compteur "depuis combien de temps ensemble"
   • Messages romantiques aléatoires
   • Quiz avec score
   • Playlist avec liens YouTube
   • Surprises aléatoires

✅ Sauvegarde Automatique
   • localStorage pour persistance des données
   • Pas besoin d'Internet
   • Données restent sur votre appareil

✅ Responsive & Mobile-Friendly
   • Fonctionne sur desktop, tablet, mobile
   • Design adaptatif
   • Touch-friendly

🆘 DÉPANNAGE RAPIDE

❌ "Le site ne s'ouvre pas"
   → Assurez-vous d'avoir les 3 fichiers (HTML, CSS, JS) dans le même dossier
   → Essayez un autre navigateur (Chrome, Firefox, Safari)

❌ "Le mot de passe ne fonctionne pas"
   → Mot de passe par défaut: amour123 (exactement)
   → Respectez majuscules/minuscules

❌ "Rien n'est sauvegardé"
   → Assurez-vous que localStorage est activé
   → Ne videz pas le cache du navigateur

❌ "Les modifications ne s'appliquent pas"
   → Sauvegardez le fichier (Ctrl+S)
   → Rafraîchissez le navigateur (Ctrl+F5)
