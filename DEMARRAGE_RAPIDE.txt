📂 VÉRIFIEZ D'ABORD QUE VOUS AVEZ CES 4 FICHIERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Vous devez avoir dans le dossier "notre univers":

  ✓ index.html          (la page web)
  ✓ style.css           (les styles)
  ✓ script.js           (la logique)
  ✓ README.md           (la documentation)
  
  (Les fichiers .txt comme celui-ci sont optionnels mais utiles)

Si tous les fichiers sont présents, passez à l'étape 1!


🎯 ÉTAPE 1 : OUVRIR LE SITE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OPTION A - Le plus simple (Double-clic) ⭐ RECOMMANDÉ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Ouvrez le dossier "notre univers"
  2. Double-cliquez sur "index.html"
  3. Le site s'ouvre dans votre navigateur par défaut
  4. Allez à l'étape 2!

OPTION B - Clic droit et ouvrir avec
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Clic droit sur "index.html"
  2. "Ouvrir avec" → Choisissez votre navigateur
     (Chrome, Firefox, Safari, Edge...)
  3. Le site s'ouvre
  4. Allez à l'étape 2!

OPTION C - Glisser-déposer
━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Ouvrez votre navigateur
  2. Glissez "index.html" sur la fenêtre
  3. Le site s'ouvre
  4. Allez à l'étape 2!


🔐 ÉTAPE 2 : SE CONNECTER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Vous verrez une belle page de login avec:
  - Le titre "🌹 Notre Univers 🌹"
  - Un champ pour entrer le mot de passe
  - Un indice (apparaît après 2 secondes)

Entrez le mot de passe par défaut: amour123

(Vous pouvez le changer plus tard dans script.js - voir CONFIGURATION.txt)


✨ ÉTAPE 3 : EXPLORER LE SITE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Une fois connecté, vous verrez:

  📱 EN HAUT : La barre de navigation
     🏠 Accueil    💌 Messages    📸 Souvenirs
     🎮 Quiz      🎵 Playlist    🎁 Surprise
     🚪 Déconnexion

  📝 AU CENTRE : Le contenu de la section active
     Commençant par la page "Accueil" avec:
     - Un message romantique
     - Le compteur "jours ensemble"
     - Boutons pour accéder aux autres sections

  Cliquez sur n'importe quel bouton pour explorer!


🎯 ÉTAPE 4 : AJOUTER DU CONTENU
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Le site est vide! Ajoutez vos premiers contenus:

💌 MESSAGES
  1. Cliquez sur "Messages"
  2. Remplissez le formulaire
  3. Cliquez "Envoyer le message"
  4. C'est sauvegardé automatiquement!

📸 SOUVENIRS
  1. Cliquez sur "Souvenirs"
  2. Entrez la date, titre, description
  3. Optionnel: Ajoutez le lien d'une photo
  4. Cliquez "Sauvegarder le souvenir"

🎵 PLAYLIST
  1. Cliquez sur "Playlist"
  2. Entrez le titre de la chanson
  3. Artiste
  4. Lien YouTube (trouvez la chanson sur YouTube et copiez l'URL)
  5. Le message que cette chanson signifie pour vous
  6. Cliquez "Ajouter à la playlist"

🎮 QUIZ
  1. Cliquez sur "Quiz"
  2. Cliquez "Commencer le Quiz"
  3. Répondez aux 5 questions
  4. Voyez votre score!

🎁 SURPRISE
  1. Cliquez sur "Surprise"
  2. Cliquez le gros bouton
  3. Une surprise aléatoire apparaît
  4. L'historique s'affiche dessous


📋 ÉTAPE 5 : PERSONNALISER (OPTIONNEL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✏️ CHANGEMENTS FACILES À FAIRE:

Changerez le mot de passe?
  → Ouvrez script.js
  → Ligne 15: PASSWORD: 'amour123'
  → Remplacez par votre mot de passe
  → Sauvegardez (Ctrl+S)
  → Rafraîchissez le site (F5)

Changer la date de début de votre histoire?
  → Ouvrez script.js
  → Ligne 16: COUPLE_START_DATE: '2023-01-14'
  → Remplacez par VOTRE DATE (format: YYYY-MM-DD)
  → Sauvegardez (Ctrl+S)
  → Rafraîchissez (F5)
  → Le compteur s'actualise automatiquement!

Pour d'autres changements, lisez CONFIGURATION.txt


💾 ÉTAPE 6 : SAUVEGARDE DE VOS DONNÉES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Les données SONT sauvegardées automatiquement
   (dans le localStorage du navigateur)

⚠️ Les données SERONT perdues si:
   - Vous videz le cache/historique du navigateur
   - Vous changez de navigateur ou d'ordinateur
   - Vous utilisez le mode "incognito/privé"

💡 CONSEIL: Sauvegardez régulièrement!

Comment sauvegarder vos données:
  1. Dans le navigateur: Faites un screenshot de chaque page
  2. Ou: Copiez les messages dans un document
  3. Ou: Gardez l'original des photos quelque part


🆘 DÉPANNAGE RAPIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Le site ne s'ouvre pas"
   → Assurez-vous d'utiliser un navigateur moderne
   → Essayez un autre navigateur
   → Vérifiez que index.html n'a pas été renommé

❌ "Le mot de passe ne fonctionne pas"
   → Mot de passe par défaut: amour123 (exactement)
   → Vérifiez majuscules/minuscules
   → Indice: "C'est le mot qui décrit nos sentiments!"

❌ "Rien n'est sauvegardé"
   → Assurez-vous que le localStorage est activé
   → Essayez de fermer le navigateur et rouvrir
   → Vérifiez que vous n'êtes pas en mode incognito

❌ "Les modifications ne s'appliquent pas"
   → Sauvegardez le fichier (Ctrl+S)
   → Rafraîchissez le navigateur (Ctrl+F5)
   → Videz le cache si nécessaire

❌ "Les images des souvenirs ne s'affichent pas"
   → L'URL doit être HTTPS (commençant par https://)
   → Vérifiez que le lien est valide
   → Sinon, une image placeholder s'affichera
