💾 DONNÉES ET SAUVEGARDE
═══════════════════════════════════════════════════════════════════════════════

OÙ LES DONNÉES SONT STOCKÉES:
   → localStorage du navigateur
   → Données complètement privées
   → Aucun envoi sur internet

QUOI EST SAUVEGARDÉ:
   ✓ Messages d'amour
   ✓ Souvenirs (titres, descriptions, URLs)

   ✓ Historique des surprises

QUOI N'EST PAS SAUVEGARDÉ:

   ✗ Les photos elles-mêmes (seulement les URLs)

SÉCURITÉ:
   ✓ Données 100% privées
   ✓ Pas de compte créé
   ✓ Pas de serveur
   ✓ Pas d'internet nécessaire

🆘 DÉPANNAGE PAR FICHIER
═══════════════════════════════════════════════════════════════════════════════

PROBLÈME: Le site ne s'affiche pas correctement
SOLUTION:
   • Assurez-vous que style.css est dans le même dossier
   • Vérifiez qu'il n'a pas été renommé
   • Essayez Ctrl+F5 (hard refresh)

PROBLÈME: Les interactions ne marchent pas (boutons, formulaires)
SOLUTION:
   • Assurez-vous que script.js est dans le même dossier
   • Vérifiez qu'il n'a pas été renommé
   • Vérifiez la console (F12) pour les erreurs
   • Essayez un autre navigateur

PROBLÈME: Les modifications de script.js ne s'appliquent pas
SOLUTION:
   • Sauvegardez le fichier (Ctrl+S)
   • Rafraîchissez le navigateur (Ctrl+F5)
   • Videz le cache (Ctrl+Shift+Delete)

PROBLÈME: Les animations sont saccadées
SOLUTION:
   • Cela dépend de votre ordinateur
   • Fermez les onglets inutiles
   • Essayez un autre navigateur
